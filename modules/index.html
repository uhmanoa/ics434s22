2. Data Wrangling - Foundations

Data wrangling is the process of transforming and mapping data from a “raw” state into a state more useful for analysis.

Mon, Jan 17 - Mon, Jan 24

3. Data Visualization

Introduction to the Matplotlib library, and some common plot types.

Mon, Jan 24 - Mon, Jan 31

4. Data Wrangling - Advanced

Preparation and cleaning, joining data, hierarchical indexing

Mon, Jan 31 - Mon, Feb 7

5. Probability Models

Introduction to probability, canonical probability distributions, sampling and parameter estimation.

Mon, Feb 7 - Mon, Feb 21

6. Hypothesis Testing

Parametric and non-parametric hypothesis testing techniques.

Mon, Feb 21 - Mon, Feb 28

7. Parameter Estimation

Parameter estimation is a way to use data to decide the values of parameters for a model.

Mon, Feb 28 - Mon, Mar 21

8. Clustering

Clustering is the task of grouping a set of objects such that objects in the same group are more similar to each other than to those in other groups.

Mon, Mar 21 - Mon, Apr 4

9. Regression

Regression is an approach to modeling the relationship between variables and responses.

Mon, Apr 4 - Mon, Apr 11

10. Time Series Data Analysis

Time series data analysis deals with the representation of data that includes a time stamp.

Mon, Apr 11 - Mon, Apr 25

11. Machine Learning

Machine learning is a field devoted to understanding and developing algorithms that use data to improve performance on a set of tasks.

Mon, Apr 25 - Wed, May 4
